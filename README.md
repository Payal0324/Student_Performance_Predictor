# 🎓 Student Performance Predictor

A simple Machine Learning web application that predicts whether a student will **Pass or Fail** based on inputs like study hours, attendance, previous marks, and assignment completion.

🚀 **Live App:** https://studentperformancepredictor-6kufqnyhuwcakgighhy3qw.streamlit.app/

---

## 📌 Project Overview

This project is a beginner-friendly implementation of a **Student Performance Prediction System** using a **Decision Tree Classifier**.

It takes student-related inputs and predicts whether the student is likely to pass or fail. This can help identify students who may need academic support early.

---

## ⚙️ Features

* 📊 Predicts **Pass / Fail**
* 🧠 Uses **Decision Tree Algorithm**
* 🌐 Interactive UI built with **Streamlit**
* ⚡ Real-time predictions based on user input
* 📁 Simple and clean project structure
